# Vacancy Pipeline

## Project Goal
Этот проект реализует простой ETL-пайплайн для сбора и анализа вакансий.

Данные о вакансиях загружаются из внешнего API, очищаются и преобразуются к единому формату, после чего сохраняются в базу данных SQLite с использованием SQLAlchemy ORM.

После загрузки данных проект позволяет выполнять аналитические запросы, такие как подсчёт количества вакансий, анализ зарплат и получение статистики по компаниям.

Проект создан в учебных целях для изучения ETL-процессов, SQLAlchemy ORM, работы с API и базовых задач Data Engineering.
## Technologies
* Python — основной язык разработки.
* SQLite — база данных для хранения вакансий и результатов обработки.
* SQLAlchemy ORM — работа с базой данных через объектно-ориентированный интерфейс.
* Requests — получение данных из внешнего API.
* BeautifulSoup — очистка и обработка HTML-данных.
* Git — контроль версий проекта.

## Project Structure
Проект разделён на несколько логических модулей:

* `etl` — получение, преобразование и загрузка данных.
* `db` — модели базы данных и настройка подключения через SQLAlchemy.
* `analytics` — аналитические запросы к базе данных.
* `tests` — тесты для проверки отдельных компонентов проекта.
* `main.py` — точка входа в приложение, объединяющая работу всех модулей.

## Database Schema
База данных содержит следующие основные таблицы:

- `companies` — компании, публикующие вакансии.
- `vacancies` — вакансии, связанные с компаниями отношением один-ко-многим.
- `tags` — теги/категории, связанные с вакансиями.
- `vacancies_tags` — промежуточная таблица для связи многие-ко-многим между вакансиями и тегами.

Связи:

- Одна компания может иметь много вакансий.
- Одна вакансия может иметь много тегов.
- Один тег может относиться ко многим вакансиям.
## Analytics
На текущий момент реализованы следующие запросы:

* Средняя зарплата.
* Максимальная зарплата.
* Минимальная зарплата.
* Общее количество вакансий.
* Количество вакансий по компаниям.
* Количество вакансий по городам.
* Топ вакансий по уровню зарплаты.
* Самый популярный тег по количеству вакансий.

## How to Run

1. Установить зависимости:
pip install -r requirements.txt

2. Запустить приложение:
python main.py
